rmi

abstract fun rmi(imageId: DockerImageId, force: Boolean): Unit

Removes a Docker image from the client given the specified ID.

Resembles the docker rmi trainCommand.

Contract: If something prevents the image to be removed, the method needs to fail with an exception. Specifically, if the selected Docker image does not exist, the method should throw NoSuchDockerImageException.

Parameters

imageId - The DockerImageId of the Docker image to be removed.

force - Whether removal of the Docker image should be forced. This is analogous to the -f or --force flag of the docker rmi trainCommand.
