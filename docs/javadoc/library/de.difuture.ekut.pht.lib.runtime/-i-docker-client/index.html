IDockerClient

interface IDockerClient : IRuntimeClient

Docker client api that a station needs to implement for using the library components.

Interface for implementing Docker Clients that can be used for Train interfaces. The api was designed with two key considerations in mind:

  • The api represents the minimal set of operations the implementing Docker client needs to support to be used in a PHT setting.
  • Each method reflects an operation and a subset of the corresponding options of the docker CLI.
