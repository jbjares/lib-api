commit

abstract fun commit(containerId: DockerContainerId, targetRepo: DockerRepositoryName, targetTag: DockerTag, author: String? = null, comment: String? = null): DockerImageId

Commits the Docker container and creates new image.

Resembles the docker commit command.

Contract: If the container selected via the DockerContainerId parameter does not exit, the method should throw an NoSuchDockerContainerException. Otherwise, if anything else the prevents the target repo to be created, the method should also throw an exception.

Parameters

containerId - The DockerContainerId of the container to be commited.

targetRepo - The DockerRepositoryName of the Docker repository to commit to.

targetTag - The DockerTag that the resulting image should be tagged with.

author -

Return
The DockerImageId that points to the newly created image.
