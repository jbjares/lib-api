push

abstract fun push(departure: ITrainDeparture<A>): Unit

Pushes the specified ITrainDeparture to this Train Registry.

Contract: The method should throw an exception if the ITrainDeparture cannot be pushed to the train registry successfully. Pushing to a ITrainDeparture that already exits and would not be updated via this push is never a failure.

Parameters

departure - The ITrainDeparture instance that should be pushed to this Train Registry.
