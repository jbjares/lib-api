ITrainRegistryClient

interface ITrainRegistryClient<A : IRuntimeClient>

Represents the api of a client to the Train Registry.

The train registry is directly dependent on a selected runtime client, as the train registry provides a way to interact with ITrainArrival.

Parameters

A - The runtime client required to use ITrainArrival instances retrieved from the implementer of the Train Registry.
