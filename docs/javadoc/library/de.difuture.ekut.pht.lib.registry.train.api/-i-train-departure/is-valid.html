isValid

abstract fun isValid(): Boolean

Checks whether this ITrainDeparture is valid, i.e. whether the result of running the algorithm can be accessed.

If this method returns false, the corresponding ITrainArrival must again run the algorithm.

Return
Whether this ITrainDeparture is still valid.
